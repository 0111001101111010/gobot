CLI

Gobot CLI (Command Line Interface) provides several tools to make RobotOps easier.

What is it?

Gobot CLI is a powerful command line interface tool that provides several tools to make RobotOps and setting up your devices to work with gobot a lot easier.

How to install

For the commands to be available everywhere the gobot module needs to be installed globally.

Here's how to do it.

go get -u github.com/hybridgroup/gobot/gobot

If you need more help to install golang click here.

How can I use it?

The Gobot CLI tools are easy to use, designed to be intuitive, and offer contextual help regarding the commands you're using. Once you have the gobot-cli module installed globally, the toolkit will be available at all times. To find all available commands, just run gobot help.

Printing a list of available commands

gobot - 

Commands:

    generate    Generates a Gobot library skeleton

Use "gobot help <command>" for more information about a command.

What Gobot CLI commands are available?

If you have only installed gobot, the list of available commands is the one printed in the section above, let's review them one by one.

gobot generate [name]
