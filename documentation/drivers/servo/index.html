Provides an interface for making servos easy to work and interact with from Gobot.

How to establish a connection and setup the driver

For Digispark:

package main

        "fmt"

        "github.com/hybridgroup/gobot-gpio"
)

func main() {

        digispark := new(gobotDigispark.DigisparkAdaptor)
        digispark.Name = "Digispark"

        servo := gobotGPIO.NewServo(digispark)
        servo.Name = "servo"

Commands:

MoveC
Moves the servo to the specified angle, angle must be an integer value between 0 and 180.
MinC
Moves the servo to the min angle 0.
CenterC
Moves the servo to the center angle 90.
MaxC
Moves the servo to the max angle 180.

Events

start
Sent when the device has been started and is ready to use.

How to use

This example moves a servo from a random angle.

Example

package main

import (
        "fmt"
        "github.com/hybridgroup/gobot"
        "github.com/hybridgroup/gobot-digispark"
        "github.com/hybridgroup/gobot-gpio"
)

func main() {

        digispark := new(gobotDigispark.DigisparkAdaptor)
        digispark.Name = "Digispark"

        servo := gobotGPIO.NewServo(digispark)
        servo.Name = "servo"

        work := func() {
                servo.InitServo()
                gobot.Every("1s", func() {
                        i := int(gobot.Rand(180))
                        fmt.Println("Turning", i)
                        servo.Move(i)
                })
        }

        robot := gobot.Robot{
                Connections: []interface{}{digispark},
                Devices:     []interface{}{servo},
                Work:        work,
        }

        robot.Start()
}
